Features

Everything you need.
Nothing you don't.

arb ships with a carefully curated set of tools so you can start working the moment you open it. No plugins to install. No config files to write.

Zero Config

Polished defaults with JetBrains Mono, Arb Dark theme, optimized macOS font rendering, and smooth animations. Open it and start working.

Built-in Shell Suite

Comes pre-loaded with Starship prompt, z smart directories, Delta for diffs, syntax highlighting, autosuggestions, and completions.

GPU Accelerated

40% smaller binary, instant startup, lazy loading, and a stripped-down GPU-accelerated rendering core. Fast and lightweight.

Lua Scripting

Full Lua scripting support for infinite customization. Configure every detail at ~/.config/arb/arb.lua

Built for AI Coding

Your AI workflow, supercharged

arb is purpose-built for developers who use Claude Code, GitHub Copilot, and AI coding tools every day. Here is how it fits your workflow.

1

Open arb, start coding

Launch arb and your shell is ready with Starship prompt, syntax highlighting, and autosuggestions. No setup required — brew install and go.

2

Run Claude Code inline

Type claude "refactor auth module" and watch AI edit your files in real time. Built-in Delta renders beautiful diffs instantly.

3

Review diffs with Delta

Every git diff is syntax-highlighted with line numbers, side-by-side view, and color-coded additions and deletions. No config needed.

4

Ship fast, stay private

Commit, push, and move on. No account needed, no telemetry, no data leaves your machine. MIT-licensed and fully open source.
